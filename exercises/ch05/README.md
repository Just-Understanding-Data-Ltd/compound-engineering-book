# Chapter 5 Exercises: The 12-Factor Agent

> Hands-on exercises for implementing production-ready agent principles.

## Exercises

| # | Title | Difficulty | Time |
|---|-------|------------|------|
| 1 | Implement Factor 1-4 | Medium | 20-25 min |
| 2 | Observability Setup | Medium | 25-30 min |
| 3 | Full 12-Factor Audit | Hard | 30 min |

## Learning Objectives

After completing these exercises, you will be able to:

- Implement core 12-factor agent principles
- Set up observability for agent workflows
- Audit existing agents against 12-factor criteria

## Prerequisites

- Claude Code CLI installed
- Understanding of agent architecture basics
- Node.js/TypeScript environment for SDK examples

## Getting Started

These exercises are more advanced. Review the 12-factor principles before beginning.
