# Chapter 4 Exercises: Writing Your First CLAUDE.md

> Hands-on exercises for creating effective CLAUDE.md files.

## Exercises

| # | Title | Difficulty | Time |
|---|-------|------------|------|
| 1 | Basic CLAUDE.md Structure | Easy | 15-20 min |
| 2 | Hierarchical Rules | Medium | 20-25 min |
| 3 | Project-Specific Instructions | Hard | 25-30 min |

## Learning Objectives

After completing these exercises, you will be able to:

- Create a well-structured CLAUDE.md file
- Implement hierarchical rule files
- Write project-specific instructions that improve agent behavior

## Prerequisites

- Claude Code CLI installed
- An existing project to document
- Completed Chapter 3 exercises

## Getting Started

Work through the exercises in order. Start with a basic CLAUDE.md and progressively enhance it.
