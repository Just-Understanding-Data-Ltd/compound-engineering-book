# Chapter 9 Exercises: Context Engineering Deep Dive

> Hands-on exercises for optimizing context usage and management.

## Exercises

| # | Title | Difficulty | Time |
|---|-------|------------|------|
| 1 | Context Budget Analysis | Medium | 20-25 min |
| 2 | Progressive Disclosure | Medium | 25-30 min |
| 3 | Context Compaction | Hard | 30 min |

## Learning Objectives

After completing these exercises, you will be able to:

- Analyze and optimize context budget allocation
- Implement progressive disclosure patterns
- Manage context rot and compaction

## Prerequisites

- Understanding of token limits and context windows
- A project large enough to stress context limits
- Familiarity with CLAUDE.md structure

## Getting Started

These exercises require measuring and analyzing context usage. Prepare to iterate.
