# Chapter 2 Exercises: Getting Started with Claude Code

> Hands-on exercises for mastering Claude Code basics.

## Exercises

| # | Title | Difficulty | Time |
|---|-------|------------|------|
| 1 | Explore a Codebase | Easy | 10-15 min |
| 2 | Iterative Development | Medium | 20-25 min |
| 3 | Tool Selection | Medium | 15-20 min |

## Learning Objectives

After completing these exercises, you will be able to:

- Use Claude Code's exploration tools effectively
- Apply the two-mode mental model (explore vs implement)
- Choose the right tool for different tasks

## Prerequisites

- Claude Code CLI installed and authenticated
- A medium-sized codebase to explore (or clone the example repo)
- Completed Chapter 1 exercises

## Getting Started

Work through the exercises in order. Each builds on concepts from the previous one.
