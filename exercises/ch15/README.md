# Chapter 15 Exercises: Model Strategy & Cost Optimization

> Hands-on exercises for optimizing AI spend and model selection.

## Exercises

| # | Title | Difficulty | Time |
|---|-------|------------|------|
| 1 | Model Selection Tree | Easy | 15-20 min |
| 2 | Cost Analysis | Medium | 25-30 min |
| 3 | Timeout Configuration | Medium | 20-25 min |

## Learning Objectives

After completing these exercises, you will be able to:

- Choose appropriate models for different task types
- Analyze and optimize AI costs
- Configure multi-layer timeouts for cost protection

## Prerequisites

- Access to multiple model tiers (Haiku/Sonnet/Opus)
- Understanding of token costs and pricing
- API usage data for analysis

## Getting Started

Start with the model selection tree to understand when to use each tier.
