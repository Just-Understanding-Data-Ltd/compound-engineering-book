# Chapter 12 Exercises: Development Workflows

> Hands-on exercises for advanced development workflow patterns.

## Exercises

| # | Title | Difficulty | Time |
|---|-------|------------|------|
| 1 | Plan Mode Workflow | Easy | 15-20 min |
| 2 | Git Worktrees Setup | Medium | 20-25 min |
| 3 | Deterministic Script | Hard | 30 min |

## Learning Objectives

After completing these exercises, you will be able to:

- Use plan mode for strategic thinking before implementation
- Set up git worktrees for parallel development
- Convert ad-hoc flows into deterministic scripts

## Prerequisites

- Git 2.15+ for worktree support
- Claude Code with plan mode enabled
- A project suitable for parallel development

## Getting Started

Plan mode provides the foundation for the other exercises in this chapter.
