# Chapter 10 Exercises: The RALPH Loop

> Hands-on exercises for implementing long-running agent patterns.

## Exercises

| # | Title | Difficulty | Time |
|---|-------|------------|------|
| 1 | Task Queue Design | Medium | 20-25 min |
| 2 | Progress File Setup | Medium | 20-25 min |
| 3 | Mini RALPH Loop | Hard | 30 min |

## Learning Objectives

After completing these exercises, you will be able to:

- Design effective task queues for agent work
- Implement progress tracking with git-based memory
- Run a basic RALPH loop for overnight development

## Prerequisites

- Understanding of agent iteration patterns
- Git repository for checkpoint commits
- Shell scripting basics for loop implementation

## Getting Started

Start with task queue design before implementing the full loop.
