# Chapter 8 Exercises: Error Handling & Debugging

> Hands-on exercises for systematic error diagnosis and recovery.

## Exercises

| # | Title | Difficulty | Time |
|---|-------|------------|------|
| 1 | Five-Point Diagnostic | Easy | 15-20 min |
| 2 | Context Debugging | Medium | 20-25 min |
| 3 | Clean Slate Recovery | Hard | 25-30 min |

## Learning Objectives

After completing these exercises, you will be able to:

- Apply the five-point error diagnostic framework
- Debug context-related issues systematically
- Recover from stuck states using clean slate technique

## Prerequisites

- A project with existing bugs or issues (or use provided examples)
- Understanding of Claude Code error patterns
- Familiarity with git for recovery scenarios

## Getting Started

Use real errors from your projects when possible. Learning to debug requires actual problems.
