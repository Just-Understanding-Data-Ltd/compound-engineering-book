# Chapter 1 Exercises: The Compound Systems Engineer

> Hands-on exercises for understanding compound engineering fundamentals.

## Exercises

| # | Title | Difficulty | Time |
|---|-------|------------|------|
| 1 | Create Your First CLAUDE.md | Easy | 10-15 min |
| 2 | Leverage Audit | Medium | 15-25 min |
| 3 | Three Levels Assessment | Medium | 20-25 min |

## Learning Objectives

After completing these exercises, you will be able to:

- Create a basic CLAUDE.md file for a project
- Identify compound leverage opportunities in your workflow
- Recognize which level of engineering you're operating at

## Prerequisites

- Claude Code CLI installed and configured
- A code project to work with (can be a simple test project)
- Basic familiarity with terminal/command line

## Getting Started

Work through the exercises in order. Each builds on concepts from the previous one.

```bash
# Start with exercise 1
cat exercises/ch01/01-first-claude-md.md
```
