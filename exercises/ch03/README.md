# Chapter 3 Exercises: Prompting Fundamentals

> Hands-on exercises for mastering effective prompting techniques.

## Exercises

| # | Title | Difficulty | Time |
|---|-------|------------|------|
| 1 | Chain-of-Thought Prompting | Easy | 15-20 min |
| 2 | Constraint-Based Prompting | Medium | 20-25 min |
| 3 | Few-Shot Examples | Medium | 20-25 min |

## Learning Objectives

After completing these exercises, you will be able to:

- Structure prompts using chain-of-thought reasoning
- Apply constraints to narrow the solution space
- Use few-shot examples effectively

## Prerequisites

- Claude Code CLI installed
- Understanding of basic prompting concepts
- A project with existing code patterns to match

## Getting Started

Work through the exercises in order. Each technique builds on the previous.
