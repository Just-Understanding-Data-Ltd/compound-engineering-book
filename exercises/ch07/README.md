# Chapter 7 Exercises: Quality Gates That Compound

> Hands-on exercises for building automated quality systems.

## Exercises

| # | Title | Difficulty | Time |
|---|-------|------------|------|
| 1 | Pre-commit Hook Setup | Easy | 15-20 min |
| 2 | CI Quality Gate | Medium | 25-30 min |
| 3 | Self-Teaching Gate | Hard | 30 min |

## Learning Objectives

After completing these exercises, you will be able to:

- Configure pre-commit hooks with Claude Code
- Set up automated code review in CI
- Build gates that improve agent behavior over time

## Prerequisites

- Git repository with hook support
- CI/CD platform access (GitHub Actions or similar)
- Completed verification ladder exercises

## Getting Started

Start with local hooks before moving to CI integration.
