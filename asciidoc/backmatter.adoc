== Appendix A: Quick Reference

=== Essential Commands

[source,bash]
----
# Start Claude Code
claude

# Run with specific prompt
claude -p "your prompt here"

# Resume session
claude --resume

# Run RALPH loop
./scripts/ralph.sh --max-hours 3
----
